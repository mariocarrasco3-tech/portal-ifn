1. Teoría del Laboratorio

Estructura básica de HTML5 y primer documento

Antes de programar, entendamos qué vamos a construir y con qué herramienta.

¿Qué es un editor de código?

Un editor de código es un programa donde se escribe el código de una página web, similar a un procesador de texto pero especializado: resalta etiquetas, numera líneas y muestra errores. Ejemplos reales que usan profesionales son Visual Studio Code o editores en línea como CodePen. En este Laboratorio Virtual tienes un editor integrado en el navegador que funciona igual: escribes código HTML a la izquierda y ves el resultado a la derecha, sin necesidad de instalar nada.

Buenas prácticas desde el primer documento
  • Todo documento HTML5 empieza con <!DOCTYPE html>, sin excepción.
  • Cada etiqueta que se abre debe cerrarse: <p>...</p>.
  • Las etiquetas van anidadas correctamente (una dentro de otra, sin cruzarse).
  • El contenido visible del usuario va siempre dentro de <body>, nunca en <head>.
